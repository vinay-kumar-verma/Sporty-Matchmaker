# Sporty 🏸

A hyperlocal sports matchmaking platform built using Flutter and Firebase.

## Features

- User Authentication
- Create and Join Games
- Skill Level Matching
- Real-time Updates
- In-app Chat
- Location Based Discovery
- Push Notifications

## Tech Stack

- Flutter
- Firebase Auth
- Cloud Firestore
- Firebase Storage
- Firebase Cloud Messaging
- Riverpod
- Go Router

## Project Structure

lib/
├── core/
├── features/
├── shared/
└── main.dart
